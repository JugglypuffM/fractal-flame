# Фрактальное пламя

Реализация алгоритма фрактального пламени с поддержкой многопоточности, log-gamma коррекции и супер-сэмплинга

```
Usage: fractal-flame [options]

  --samples <value>         Количество начальных точек
  --iterations <value>      Количество итераций отображения точки
  --threads <value>         Количество потоков, используемых для генерации изображения,
                            не может быть больше количества потоков компьютера
  --width <value>           Ширина результирующего изображения
  --height <value>          Высота результирующего изображения
  --sampling-factor <value> Фактор супер-сэмплинга, множитель размера изображения
                            на время генерации - после генерации изображения
                            сожмется до результирующего
  --xMin <value>            Минимальное значение x компоненты области отрисовки функции
  --xMax <value>            Максимальное значение x компоненты области отрисовки функции
  --yMin <value>            Минимальное значение y компоненты области отрисовки функции
  --yMax <value>            Максимальное значение x компоненты области отрисовки функции
  --gamma <value>           Параметр для гамма-коррекции изображения
  --affine-count <value>    Количество афинных преобразований,
                            недостающее количество будет сгенерировано случайно
  --variation-count <value> Количество нелинейных преобразований,
                            недостающее количество будет сгенерировано случайно
  --affine <value>          Array of the form r,g,b,a,b,c,d,e,f
  --variation <value>       Номер нелинейного преобразования соответствующий:
                             1 - Sinusoidal
                             2 - Spherical
                             3 - Swirl
                             4 - Horseshoe
                             5 - Polar
                             6 - Spiral
                             7 - Handkerchief
                             8 - Heart
                             9 - Disc
                             10 - Diamond
                             11 - Ex
                             12 - Julia
                             13 - Waves
                             14 - Fisheye
                             15 - Bubble
                             16 - Exponential
                             other - Linear
  --symmetry <value>        Параметр симметрии, количество повротов точки относительно центра
  --file <value>            Путь для сохранения изображения, принимает в частности название файла
```
